# Student Performance Prediction

## 📌 Project Description
A Machine Learning web application that predicts student performance based on study-related parameters using Python, Flask, and Scikit-learn.

## 🚀 Features
- Predict student performance
- User-friendly web interface
- Machine Learning model
- Flask backend
- Responsive UI

## 🛠 Technologies Used
- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS

## 📂 Project Structure
Student_Performance_Prediction/
│── app.py
│── train_model.py
│── dataset/
│── models/
│── templates/
│── static/

## ▶️ Run the Project

pip install -r requirements.txt

python app.py

http://127.0.0.1:5000
